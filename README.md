# 🎓 Grade Central CLI

A simple command-line application to manage student records, built using Python.

---

🚀 Features

- Add grades to existing students
- Remove students from the system
- Calculate and display average grades
- Basic admin authentication

---

🛠️ Tech Stack

- Python 3
- Built-in "statistics" module

---

▶️ How to Run

python grade_central.py

⚠️ Limitations

- Data is not saved (resets after program exit)
- No input validation for grade ranges
- Authentication is not secure (hardcoded credentials)
